How we display Scripture

English and original-language Bible text in this app is fetched live from the public bible.helloao.org API — not memorized or invented by AI. When a verse appears in chat, the translation abbreviation is shown alongside the reference (for example, John 3:16 (BSB)).

This app does not display ESV, NASB, NKJV, NIV, NLT, or other copyrighted translations without publisher permission. English quotations use only the translations listed below.
